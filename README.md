# nihilo

An pusher server implementation compatible with Pusher client libraries written in Go.

## Features

- Public Channels
- Private Channels
- Presence Channels
- Web Hooks
- Client events
- Complete REST API
- Easy installation
- A single binary without dependencies
- Easy configuration
- Protocol version 7
- Multiple apps in the same instance
- Drop in replacement for pusher server
